# Agent SOP Skills

The following skills are available globally. Suggest the appropriate one when the context matches:

- When I ask to implement a feature, fix a bug, or work on a coding task, suggest using `/code-assist`.
- When I describe a rough idea or want to plan a project from scratch, suggest using `/pdd`.
- When I want to generate task files or break down work into structured tasks, suggest using `/code-task-generator`.
- When I want to analyze or document a codebase, generate AGENTS.md, README, or other docs, suggest using `/codebase-summary`.
- When I want to evaluate or test an AI agent's performance, suggest using `/eval`.
